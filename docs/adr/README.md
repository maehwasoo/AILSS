# ADR (Architecture Decision Record)

이 폴더는 “왜 이 구조를 선택했는지”를 기록하는 ADR(Architecture Decision Record) 문서들을 보관해요.

## 규칙

- 파일명: `NNNN-짧은-제목.md`
- 한 ADR에는 “결정 1개”만 담아요
- 상태(status): `draft` → `accepted` → (필요 시) `superseded`

## 템플릿(template)

```
# NNNN. 제목

status: draft

## 맥락(context)
- ...

## 결정(decision)
- ...

## 결과(consequences)
- 장점
- 단점/리스크

## 대안(alternatives)
- ...
```
